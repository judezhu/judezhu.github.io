Connect Raspberry with your Laptop

For Ubuntu users, here are some tips to connect with your pi with a standard ethenet cable.

SSH to your pi

  • connect the raspberry pi to the computer using an ethernet cable in ubuntu, go to Settings - Network - Wired - IPv4 -> scroll dropbown to Share with other, this will give internet to the Raspberr pi

      ifconfig -a

  • find address in eth0. E.g. 10.42.0.1

      nmap 10.42.0/24

  • use the port you find to connect to the raspberry pi

      ssh pi@10.42.0.41

VNC to your pi

For users, who wants to remotely see the desktop of Raspberry pi, it’s necessary to set up the VNC.

  • First we need to install the VNC server on raspberry pi:

       sudo apt-get install tightvncserver

  • Next, start the VNC server on the Raspberry Pi. Adjust the geometry paramater to your desired display size.

       vncserver :1 -geometry 1024x600 -depth 16 -pixelformat rgb565

    You may be asked for setting up password.

  • Now, we can finally connect to the Pi with VNC. Back on the Ubuntu machine, install the VNC viewer client:

        sudo apt-get install xtightvncviewer

  • Then connect to the running VNC server:

    vncviewer 10.42.0.41:5901

    Here 5901 is the default port for the VNC server running on your pi.

  • To stop the VNC viewer, just close the application. To stop the VNC server, issue the following command (on the Raspberry Pi):

        vncserver -kill :1
