Tips about using Git

11May
  1. Set up your global username and email address.

Some git beginners forget to set 

$ git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com

If you don’t do so, you will find that you will not shown as contributor in any of the Github repos, even though you commit code.

Create your personal blog in 10 mins

9May

I believe many people like me hope that they can have their own personalized blog.There are definitely many choices.

1. Choose existing blog.

You can use existing blog services such as bloggers, Yahoo/blog, Google/Site, if you don’t have high expectation of your blog. And also you want your small website to start to run ASAP. Please go with this direction.

2. Choose wordpress.

Wordpress has been deeply integrated with domain providers such as godaddy. Ittakes probably 5 mins to build a wordpress powered websites if you pay for the hosting services.

3. Choose Hexo

Here are 3 reasons for choosing Hexo to be your blog framework:

  • Markdown support
  • Powered by Node.js
  • Easy deployment to different cloud platform
